# Lumina Vids - рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ (рЃЦрЃљрЃарЃЌрЃБрЃџрЃљрЃЊ)

## ­ЪЊІ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљ

**Lumina Vids** рЃљрЃарЃўрЃА **рЃЋрЃўрЃЊрЃћрЃЮ рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃў** (video generator), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА (images) рЃљрЃЦрЃфрЃћрЃЋрЃА рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃљрЃЊ (videos) рЃАрЃЮрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃЏрЃћрЃЊрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА (social media).

---

## ­ЪЈЌ№ИЈ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў (рЃЦрЃљрЃарЃЌрЃБрЃџрЃў рЃЌрЃљрЃарЃњрЃЏрЃљрЃюрЃћрЃЉрЃўрЃЌ)

### Frontend (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў)
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **React** | рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃљрЃАрЃљрЃгрЃДрЃЮрЃЉрЃљрЃЊ. рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА, рЃарЃЮрЃЏ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃў "рЃфрЃЮрЃфрЃ«рЃљрЃџрЃў" рЃўрЃДрЃЮрЃА |
| **TypeScript** | JavaScript-рЃўрЃА рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ рЃбрЃўрЃърЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃўрЃЌ. рЃљрЃа рЃўрЃФрЃџрЃћрЃЋрЃљ рЃерЃћрЃфрЃЊрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃерЃЋрЃћрЃЉрЃљрЃА |
| **Vite** | рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃљрЃАрЃљрЃгрЃДрЃЮрЃЉрЃљрЃЊ (build tool). рЃАрЃгрЃарЃљрЃцрЃўрЃљ рЃЊрЃљ рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ |
| **Tailwind CSS** | CSS рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃў - рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ. рЃБрЃЎрЃЋрЃћ рЃЏрЃќрЃљ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ |

### State Management (рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ)
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **Zustand** | рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ. рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃА (state) рЃЏрЃЌрЃћрЃџ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃерЃў |

### Backend (рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃюрЃљрЃгрЃўрЃџрЃў)
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **Node.js** | JavaScript рЃњрЃљрЃарЃћрЃЏрЃЮ рЃАрЃћрЃарЃЋрЃћрЃарЃќрЃћ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА |
| **Express** | рЃЋрЃћрЃЉ-рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃў API-рЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃЏрЃЮрЃЌрЃ«рЃЮрЃЋрЃюрЃћрЃЉрЃА (requests) |

### Video Generation (рЃЋрЃўрЃЊрЃћрЃЮрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ)
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **Python** | рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ рЃЋрЃўрЃЊрЃћрЃЮ рЃЊрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА |
| **MoviePy** | Python рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ рЃЊрЃљ рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА |
| **ImageIO** | рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА/рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃўрЃА рЃгрЃљрЃАрЃљрЃЎрЃўрЃЌрЃ«рЃљрЃЊ/рЃЊрЃљрЃАрЃљрЃгрЃћрЃарЃљрЃЊ |
| **FFmpeg** | рЃЋрЃўрЃЊрЃћрЃЮ рЃЎрЃЮрЃЊрЃћрЃЎрЃў - рЃљрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃА рЃЊрЃљ рЃЊрЃћрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃА рЃЋрЃўрЃЊрЃћрЃЮ рЃцрЃљрЃўрЃџрЃћрЃЉрЃА |

### Desktop App (рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃЏрЃљрЃњрЃўрЃЊрЃўрЃА рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ)
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **Electron** | рЃ«рЃћрЃџрЃАрЃљрЃгрЃДрЃЮ desktop рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ (Windows/Mac). рЃљрЃћрЃарЃЌрЃўрЃљрЃюрЃћрЃЉрЃА рЃЋрЃћрЃЉ-рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃА |

### рЃАрЃ«рЃЋрЃљ рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў
| рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|------------|--------|
| **React Router** | рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃќрЃљрЃфрЃўрЃљ - рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ |
| **Lucide React** | рЃ«рЃљрЃбрЃБрЃџрЃћрЃЉрЃўрЃА (icons) рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ |
| **Multer** | рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА middleware |
| **Archiver** | ZIP рЃцрЃљрЃўрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ |
| **UUID** | рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў ID-рЃћрЃЉрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃў |

## ­Ъјг рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў (рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА)

### 1. рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ
- **Drag & Drop** - рЃњрЃљрЃЊрЃЏрЃЮрЃбрЃљрЃюрЃљ рЃЊрЃљ рЃЊрЃљрЃњрЃЊрЃћрЃЉрЃљ
- **File Picker** - рЃцрЃљрЃўрЃџрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ
- рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ рЃћрЃарЃЌрЃЊрЃарЃЮрЃБрЃџрЃљрЃЊ

### 2. рЃЋрЃўрЃЊрЃћрЃЮрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃў
| рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў | рЃљрЃдрЃгрЃћрЃарЃљ |
|-----------|--------|
| **FPS** | рЃЎрЃљрЃЊрЃарЃћрЃЉрЃў рЃгрЃљрЃЏрЃерЃў (рЃцрЃарЃћрЃўрЃЏрЃћрЃЉрЃў рЃгрЃљрЃЏрЃерЃў) |
| **Duration** | рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃ«рЃљрЃюрЃњрЃарЃФрЃџрЃўрЃЋрЃЮрЃЉрЃљ рЃгрЃљрЃЏрЃћрЃЉрЃерЃў |
| **Transition** | рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА рЃерЃЮрЃарЃўрЃА |
| **Transition Duration** | рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃўрЃА рЃ«рЃљрЃюрЃњрЃарЃФрЃџрЃўрЃЋрЃЮрЃЉрЃљ |
| **Music** | рЃцрЃЮрЃюрЃБрЃарЃў рЃЏрЃБрЃАрЃўрЃЎрЃљ |
| **Volume** | рЃЏрЃБрЃАрЃўрЃЎрЃўрЃА рЃ«рЃЏрЃўрЃА рЃФрЃљрЃџрЃљ |

### 3. рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃцрЃЮрЃарЃЏрЃљрЃбрЃћрЃЉрЃў (4 рЃфрЃљрЃџрЃў)
| рЃцрЃЮрЃарЃЏрЃљрЃбрЃў | рЃќрЃЮрЃЏрЃљ | рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ |
|---------|------|-----------|
| **9:16** | рЃЋрЃћрЃарЃбрЃўрЃЎрЃљрЃџрЃБрЃарЃў | TikTok, YouTube Shorts, Reels |
| **1:1** | рЃЎрЃЋрЃљрЃЊрЃарЃљрЃбрЃў | Instagram Post |
| **4:5** | рЃърЃЮрЃарЃбрЃарЃћрЃбрЃў | Facebook Portrait |
| **16:9** | рЃџрЃљрЃюрЃЊрЃерЃљрЃцрЃбрЃў | YouTube Landscape |

### 4. рЃбрЃарЃљрЃюрЃќрЃўрЃфрЃўрЃћрЃЉрЃў (20+ рЃћрЃцрЃћрЃЦрЃбрЃў)

#### рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў:
- **Cut** - рЃЏрЃЎрЃЋрЃћрЃЌрЃарЃў рЃГрЃарЃљ (рЃАрЃБрЃарЃљрЃЌрЃў рЃБрЃфрЃћрЃЉ рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ)
- **Fade** - рЃарЃЉрЃўрЃџрЃў рЃњрЃљрЃЊрЃљрЃАрЃЋрЃџрЃљ (рЃћрЃарЃЌрЃў рЃЦрЃарЃћрЃЉрЃљ, рЃЏрЃћрЃЮрЃарЃћ рЃЕрЃюрЃЊрЃћрЃЉрЃљ)

#### рЃАрЃџрЃљрЃўрЃЊрЃћрЃЉрЃў:
- **Slide Left/Right/Up/Down** - рЃАрЃБрЃарЃљрЃЌрЃў рЃњрЃљрЃЊрЃљрЃљрЃЊрЃњрЃўрЃџрЃЊрЃћрЃЉрЃљ рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃБрЃџ рЃЏрЃўрЃЏрЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃўрЃЌ

#### рЃќрЃБрЃЏрЃў:
- **Zoom In** - рЃЏрЃЮрЃљрЃ«рЃџрЃЮрЃћрЃЉрЃљ (рЃерЃўрЃњрЃюрЃўрЃЌ рЃерЃћрЃГрЃарЃљ)
- **Zoom Out** - рЃЊрЃљрЃерЃЮрЃарЃћрЃЉрЃљ (рЃњрЃљрЃарЃћрЃЌ рЃњрЃљрЃЏрЃЮрЃГрЃарЃљ)

#### рЃњрЃљрЃЊрЃљрЃгрЃћрЃЋрЃљ (Wipe):
- **Wipe Left/Right/Up/Down** - рЃћрЃарЃЌрЃў рЃАрЃБрЃарЃљрЃЌрЃў "рЃњрЃљрЃўрЃгрЃЏрЃћрЃюрЃЊрЃћрЃЉрЃљ" рЃЊрЃљ рЃЏрЃћрЃЮрЃарЃћ рЃњрЃљрЃЏрЃЮрЃЕрЃюрЃЊрЃћрЃЉрЃљ

#### рЃАрЃърЃћрЃфрЃўрЃљрЃџрЃБрЃарЃў рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃў:
- **Pixelate** - рЃърЃўрЃЦрЃАрЃћрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (рЃАрЃБрЃарЃљрЃЌрЃў рЃърЃўрЃЦрЃАрЃћрЃџрЃћрЃЉрЃљрЃЊ рЃўрЃерЃџрЃћрЃЉрЃљ)
- **Ripple** - рЃбрЃљрЃџрЃдрЃљ (рЃгрЃДрЃџрЃўрЃА рЃбрЃљрЃџрЃдрЃўрЃА рЃћрЃцрЃћрЃЦрЃбрЃў)
- **Page Curl** - рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃљрЃЌрЃЋрЃџрЃљ (рЃњрЃљрЃЊрЃљрЃцрЃБрЃарЃфрЃЋрЃџрЃљ)
- **Circle Open/Close** - рЃгрЃарЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ/рЃЊрЃљрЃ«рЃБрЃарЃЋрЃљ
- **Spin In/Out** - рЃЉрЃарЃБрЃюрЃЋрЃљ
- **Fly In/Out** - рЃерЃћрЃЏрЃЮрЃцрЃарЃћрЃюрЃљ/рЃњрЃљрЃцрЃарЃћрЃюрЃљ

Рєњ State (Zustand/React State) рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃА

### 3. рЃљрЃгрЃЋрЃћрЃЉрЃљ "Generate" рЃдрЃўрЃџрЃљрЃЎрЃА

### 4. Backend-рЃў рЃўрЃдрЃћрЃЉрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА
Рєњ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА `uploads/` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў

### 5. Backend-рЃў рЃБрЃерЃЋрЃћрЃЉрЃА Python рЃАрЃЎрЃарЃўрЃърЃбрЃА

### 6. Python рЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃА
Рєњ рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃбрЃарЃљрЃюрЃќрЃўрЃфрЃўрЃћрЃЉрЃА, рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃЏрЃБрЃАрЃўрЃЎрЃљрЃА
Рєњ рЃљрЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃА FFmpeg-рЃўрЃЌ

### 7. рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ
Рєњ `outputs/` рЃАрЃљрЃЦрЃљрЃдрЃљрЃџрЃЊрЃћрЃерЃў 4 рЃЋрЃўрЃЊрЃћрЃЮ рЃцрЃљрЃўрЃџрЃў

### 8. Frontend-рЃў рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА рЃърЃарЃЮрЃњрЃарЃћрЃАрЃА
